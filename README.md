This is a full-stack User Registration System built using:
- Angular (Frontend)
- ASP.NET Core Web API (Backend)

The application allows users to register by providing:
✔️ First Name  
✔️ Last Name  
✔️ Date of Birth  
✔️ Email  
✔️ Password (Strong Password Validation)  
✔️ Phone Number  

Data is validated at both frontend and backend.

---

  Features:-
- Responsive UI (Bootstrap)
- Form Validation
- Password Strength Validation
- DOB Format Validation (DD-MM-YYYY)
- REST API Integration
- CORS Enabled
- Swagger API Documentation
